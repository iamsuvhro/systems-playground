Notes Day 3:
All the notes of day 3 focusing on Pointer.

Concept:

1. In C to check or get the address of a variable need to use '&' (e.g. &x)
2. To define a pointer in C need to use *p 

1. Defining (The "Badge")
int *p;
  - Meaning: "Create a variable named p that is allowed to hold addresses."
  - The Star: Tells the computer p is not a normal integer.

2. Linking (The "Connection")
p = &x;
  - Meaning: "Take the address of x and save it inside p."
  - The Star: NO STAR.
  - Why? Because p holds addresses. &x is an address. They match perfectly.

3. Using (The "Action")
Once you have linked them, if you want to actually change the value of 10 to 20 using the pointer, that is when you bring the star back.
*p = 20;
   - Meaning: "Travel to the address inside p, and change the value there to 20."
   - The Star: YES. The star acts as the "key" to open the door and touch the data.
